# 被動元件產業（台股觀點）

台灣是全球被動元件第二大產地（僅次於日本），在中高階 MLCC、車用電感、
高頻 chip resistor 都有代表廠商。產業特性：強週期、重資本、寡占。

## Constituents

見 [[themes/passive-components]] 完整名單。本頁聚焦在產業層級的觀察軸：
財報結構、競爭格局、地緣風險。

## Key Players (Global)

- **日廠**：Murata、TDK、Taiyo Yuden — 高階 MLCC 龍頭
- **韓廠**：Samsung Electro-Mechanics
- **台廠**：[[tickers/2327]]、[[tickers/2492]]、[[tickers/3026]]、[[tickers/2456]]
- **陸廠**：風華高科、三環、宇陽 — 消費性 MLCC 殺價競爭

## Competitive Moats

- **車規 / 工規認證** — 高階 MLCC（X8R、X8L）認證至少 2-3 年，准入門檻高
- **小型化技術** — 0201、01005 尺寸（每縮一級價格 30-50%）
- **高容量** — 100µF 以上高容值 MLCC
- **高頻電感** — AI server 用 0.X µH 高頻電感
- **自有原料** — 鈦酸鋇粉、銅鎳漿料垂直整合

## Cyclical Patterns

歷史週期典型走法：

1. **底部訊號**：通路商庫存週轉天數降到 60-70 天，原廠 cap-ex 縮減
2. **上升期初**：車用 MLCC 漲價傳聞 → 報價函發出 → 股價先漲
3. **上升期中**：消費性 MLCC 跟進，財報毛利率反映
4. **頂部訊號**：通路商存貨天數爆衝 120+ 天，原廠擴產 capex 大幅上修
5. **下行**：價格下殺 + 一次性存貨損失打入

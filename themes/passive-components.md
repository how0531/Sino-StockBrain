# 被動元件 (Passive Components)

電子產品中除半導體之外的關鍵元件 — MLCC（積層陶瓷電容）、晶片電阻、
電感、鋁質電容、薄膜電容等。台廠在中高階 MLCC、車用電感供應鏈具國際地位。

## Sub-segments

- **MLCC**（積層陶瓷電容）— 用量最大、價格波動最劇烈，車規 MLCC 高毛利
- **晶片電阻** — 走量產品，毛利穩定
- **電感** — 高頻電感切入 AI server 用電源轉換
- **鋁質電解電容** — 工業 / 家電
- **薄膜電容** — EV 動力系統、太陽能逆變器

## Cycle Drivers

被動元件是出名的「重資本 + 強週期」產業。週期判斷靠：

- **車用 MLCC 漲價傳聞** — 領先指標，通常領先股價 1-2 個月
- **手機 + PC 庫存週轉天數** — 落後指標，反映消費性需求底部
- **AI server 用高頻電感拉貨** — 2024 起的結構性成長軸
- **原廠資本支出** — Murata、TDK、Samsung Electro 的 capex 指引
- **庫存週轉天數** — 自台廠財報季度揭露
